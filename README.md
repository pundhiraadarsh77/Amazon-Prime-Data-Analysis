# Amazon Prime Data Analysis

## Project Overview
This project focuses on analyzing Amazon Prime content data to uncover trends in content performance, ratings, genres, and audience preferences.

## Objective
To derive meaningful insights that can help improve content strategy and enhance platform growth.

## Tools & Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Majority of content falls within the moderate rating range (5–8)
- Drama, Comedy, and Thriller are the most dominant genres
- Limited content in genres like War, Sport, and Reality
- No strong relationship between runtime and ratings
- US contributes the highest number of titles

## Conclusion
The analysis shows that content quality plays a more important role than duration. Increasing genre diversity and geographical representation can help improve audience engagement and drive platform growth.
